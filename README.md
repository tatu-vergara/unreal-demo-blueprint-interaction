# Unreal Engine 5 Blueprint Interaction Demo

This project is a small Unreal Engine 5 prototype built using Blueprints, to show basic knowledge.

## Features

* Third-person character movement (from the demo)
* Interaction system using overlap detection
* Input handling (E key)
* UI prompt when in range
* Dynamic material switching

## Controls

* WASD → Move
* Mouse → Camera
* E → Interact

## Tech

* Unreal Engine 5
* Blueprints
* UMG (UI)

## Purpose

This project demonstrates basic Unreal Engine workflow and Blueprint scripting for gameplay interaction systems.
